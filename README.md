🔄 Estrutura do Scrum
📅 Sprints

O trabalho é dividido em ciclos chamados Sprints, onde cada ciclo possui metas específicas a serem entregues.

📌 Reuniões
Daily Scrum: alinhamento diário rápido sobre o progresso
Sprint Planning: planejamento das tarefas da Sprint
Sprint Review: revisão do que foi desenvolvido
Sprint Retrospective: análise para melhorias do time
🎯 Objetivo do Time

Desenvolver soluções eficientes, organizadas e de qualidade, utilizando boas práticas de desenvolvimento e colaboração em equipe.

🚀 Valores do Time
Comunicação clara
Responsabilidade compartilhada
Evolução constante
Compromisso com qualidade
